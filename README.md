🕒 Analog Clock Project
[view the clock] (https://digitalclockap2026.netlify.app/)

A beautiful real-time analog clock built using HTML, CSS, and JavaScript.
The clock displays animated second and minute rings with a modern dark UI design.

✨ Features
🕒 Real-time clock updates every second
🔵 Animated circular second & minute indicators
🎯 Dynamic hour and minute display
🎨 Clean dark UI design
⚡ Lightweight (no libraries required)
💻 Fully responsive structure (basic desktop support)
📁 Project Structure
clock-project/
│
├── index.html     # Main HTML file
├── style.css      # Styling for the clock
└── script.js      # Clock logic (JavaScript)
🚀 How to Run the Project
Download or clone the project
Make sure all files are in the same folder:
index.html
style.css
script.js
Open index.html in your browser
Enjoy your live analog clock ⏰
🧠 How It Works
JavaScript uses Date() to get the current time
Seconds and minutes are converted into rotation degrees:
360° / 60 = 6° per step
CSS variables (--dRotate) control smooth rotation animations
60 spike elements are dynamically created for clock markings
🛠️ Technologies Used
HTML5
CSS3 (Custom properties + animations)
Vanilla JavaScript (No frameworks)
